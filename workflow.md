# Git Workflow

## Start a Project

```
# Create Folder and enter
git init
```
## Add a Remote Repository as Default

```
git remote add origin {repository}
# Set as Default Upstream
git push --set-upstream origin master
```

## Add Readme.md

```
vim readme.md
git add readme.md
git commit -m "Added Readme.md"
git push
```
